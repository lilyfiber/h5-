# active

> 下拉随机抽取符

## Build Setup

``` bash

# 推荐使用cnpm i安装
npm i or cnpm i

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test

# 框架基于 vue + vant / vant提供基础的UI组件
# 活动发布都在本项目配置
├── build                      // 构建配置
├── config                     // 基础配置
├── src                        // 开发文件
│   ├── assets                 // 资源
│   ├── common                 // 公用文件
│   ├── components             // 组件
│   ├── router                 // 路由
│   ├── store                  // store
│   └── view                   // 模板
│         ├── 404              // 统一报错页面
│         ├── new              // 新活动模板
│         └── teacher          // 教师节活动
├── static                     // 静态资源
│   ├── images                 // 图片
│   ├── js                     // js
│   └── css                    // css
├── test                       // 测试
├── .babelrc                   // babel-loader 配置
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
├── .nvmrc                     // 环境配置文件
├── index.html                 // 入口模板
└── package.json               // package.json

```
